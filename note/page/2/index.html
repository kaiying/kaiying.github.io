isVueInstance is deprecated and will be removed in the next major version.

description

完整錯誤訊息：

錯誤訊息：isVueInstance 將在下一個主要版本棄用。

棄用原因：這個方法以長遠來看，並沒有存在的必要，用 findComponent 可以得到 vueWrapper ，這也代表他是 vue instance。

根據棄用原因，我試了這個方法，拿到 vueWrapper，然後就卡住了，我要怎麼辨別 vueWrapper？ typeof 的結果是 object ，也不可能生成一個一模一樣的內容來當測試結果。

官網文件的 update to V1 給了一個比較好的解法：wrapper.vm。
